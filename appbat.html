Multi launch is useful for creating an icon that can run multiple programs or open multiple files.
Set launch is useful for creating an icon that can run many programs, or open many files similar to multi launch, except that set launch can set conditions for running programs or opening files.

The condition for set launch is "After", which functions to run the program after clicking "Next" on the application or pressing any button to continue on the terminal (if running through an icon). the next condition is "Fail", which is useful for running the next program if the previous program failed to run. The last condition is "None", which means running the program at the same time.
Search is useful for finding exec commands on Linux, or .exe files on Windows
New bookmarks are useful for storing filepaths or website addresses, along with titles, groups, and comments from bookmarks.
